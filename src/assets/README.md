# Assets Folder

## Required Image

Please add a river image named `backdrop.jpg` to this folder.

### Image Requirements:
- **Filename**: `backdrop.jpg`
- **Type**: JPEG image
- **Recommended size**: 1920x1080 or higher
- **Content**: River or water-related background image
- **Purpose**: Used as a subtle backdrop for the app

### Where to find river images:
- [Unsplash](https://unsplash.com/s/photos/river) - Free high-quality images
- [Pexels](https://www.pexels.com/search/river/) - Free stock photos
- [Pixabay](https://pixabay.com/images/search/river/) - Free images

### How to add:
1. Download a river image
2. Rename it to `backdrop.jpg`
3. Place it in this folder (`src/assets/`)
4. Restart the app

The image will appear as a subtle background (30% opacity) behind the app content.
